Hash

  • key--hash-->index
      • hash('it') = 3, hash('bob') = 4
    1. Compute hash function
    2. Equality test: 2 keys are equal
    3. Collision resolution: Algorithm and data structure to handle two keys hash to same array index
      1. separate chain
      2. linear probing

    Space Time Tradeoff

      No space limitation: large array and use key as index
      No time limitation: hash to same index and perform sequential search for collision resolution

    Hash function

      Goal: scramble keys uniformly to produce table index
      efficiently computable
      each table index equally likely for each key
    http://algs4.cs.princeton.edu/34hash/
